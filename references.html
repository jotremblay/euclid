Loading your own data

The application is written in standard R/Shiny, its source code is publically available here

The application can be easily adapted to work with any data in CSV form, that has a column to represent the date, a column to represent an athlete and a series of numeric columns to represent "load". The code for reading data in is here

Please feel free to get in touch with me at @walmc if you want help loading your data in.
